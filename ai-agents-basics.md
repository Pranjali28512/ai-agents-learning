# AI Agents Basics

## What is an AI Agent?

An AI agent is a system that can perceive its environment, make decisions, and take actions to achieve a goal.

Unlike a simple chatbot, an AI agent can:
- Reason
- Plan steps
- Use tools
- Perform multiple actions autonomously

Example:

A chatbot answers questions.

An AI agent can:
1. Search a database
2. Call APIs
3. Analyze results
4. Give an intelligent response.

---

## Core Components of an AI Agent

1. **LLM (Large Language Model)**  
   Provides reasoning and understanding.

2. **Tools**  
   External systems the agent can use.

   Examples:
   - Search APIs
   - Databases
   - Calculators
   - Web scraping tools

3. **Memory**  
   Allows the agent to remember past interactions.

4. **Planning System**  
   Helps the agent break down tasks into smaller steps.
